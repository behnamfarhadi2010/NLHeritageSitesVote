
# NL Heritage Sites Vote 🏛️

A web application that allows users to explore and vote for Newfoundland and Labrador's most significant heritage sites. Built to celebrate and preserve the cultural history of NL through community engagement.

## ✨ Features

- Browse Newfoundland & Labrador heritage sites
- Vote for your favourite historic locations
- View real-time vote counts and rankings
- Responsive design for mobile and desktop
- Fast, modern UI built with React + Vite

## 🚀 Getting Started

### Prerequisites

- Node.js v20 or higher
- npm v10 or higher

### Installation

```bash
# Clone the repository
git clone https://github.com/behnamfarhadi2010/NLHeritageSitesVote.git

# Navigate into the project
cd NLHeritageSitesVote

# Install dependencies
npm install

# Start the development server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.
